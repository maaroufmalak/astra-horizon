# Astra Horizon

Astra Horizon is a fictional demo landing page for a luxury space exploration company. It was created as a portfolio project and is not connected to a real company, product, or service.

## Project Overview

This static website presents a cinematic brand concept for reusable launch systems, orbital habitats, and future space travel. The project focuses on visual storytelling, responsive front-end design, smooth interactions, and a premium landing page experience.

## Features

- Responsive mobile-first layout
- Cinematic hero section with custom visual assets
- Scroll reveals, counters, navigation interactions, and motion details
- Semantic HTML structure
- No framework or build step required

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript

## Run Locally

Open `index.html` directly in a browser, or serve the folder with a static server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

This is a fictional brand concept made for demonstration purposes only. All company messaging, mission claims, and business details are invented for the portfolio presentation.
